Soil constitutive models

The constitutive_models package contains modules that describe soil deformation in response to changes in stress and or time.

Modules:

epus Elasto-Plastic for Unsaturated Soils, adapted from VB code in Phd thesis of Pham (2005).
swcc Some soil water characteristic curves (SWCC) for unsaturated soil
velocity_hydraulic_gradient Relationships between velocity and hydraulic gradient
void_ratio_permeability Relationships between permeability and void-ratio
void_ratio_stress Relationships between void-ratio and effective stress.
