geotecha¶

A software package for geotechnical engineering

geotecha is a software package for geotechnical engineering written by Dr. Rohan Walker. There are numerous sub packages that may be useful outside of a geotechnical context. The geotecha project began in 2013.

Sub-packages summary:

beam_on_foundation Beams on foundations
consolidation Soil consolidation
constitutive_models Soil constitutive models
inputoutput General input/output operations
mathematics Mathematical operations/functions
piecewise Manipulation and interpolation of piecewise data/functions
plotting Generation and formatting of matplotlib plots
speccon Solving soil consolidation PDE’s using the spectral method
